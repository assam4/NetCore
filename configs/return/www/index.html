return

The return directive issues an HTTP redirect. Requests to /old are permanently redirected to /new/.

Config summary

LocationDirective
/oldreturn 301 http://localhost:8080/new/
/newserves new/index.html

Try it

Visit /old (redirects)

Terminal tests

curl -i  http://localhost:8080/old    # 301 Location: /new/
curl -iL http://localhost:8080/old    # follows redirect, shows /new/ page
