client_max_body_size

Upload body size is limited to 100 KB. Requests exceeding this return 413 Content Too Large.

Upload a file

Files under 100 KB will be accepted; files over 100 KB will be rejected.

Terminal tests

# Small file — should succeed (201)
dd if=/dev/urandom bs=1k count=50 | base64 > /tmp/small.txt
curl -i -F "file=@/tmp/small.txt" http://localhost:8080/upload

# Large file — should fail (413)
dd if=/dev/urandom bs=1k count=200 | base64 > /tmp/large.txt
curl -i -F "file=@/tmp/large.txt" http://localhost:8080/upload
