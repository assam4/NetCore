The Set-Cookie header is set by the server on your first visit.
Subsequent requests carry it back automatically.

How to verify in terminal:

# First request — server sets Set-Cookie
curl -c cookies.txt -v http://localhost:8080/session.html 2>&1 | grep -i 'set-cookie\|cookie'

# Second request — browser/curl sends Cookie header back
curl -b cookies.txt -v http://localhost:8080/session.html 2>&1 | grep -i 'set-cookie\|cookie'
